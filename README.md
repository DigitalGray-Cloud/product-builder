# Fortune Teller (Zodiac)

A mystical web application that reveals your daily and yearly fortune based on your Chinese Zodiac sign.

## 🔮 Live Demo
[**Click here to view the live site**](https://digitalgray-cloud.github.io/product-builder/)

## ✨ Features
- **Multilingual:** Instantly switch between English (EN) and Korean (KO).
- **Zodiac Insight:** Discover your animal sign and read personalized fortunes.
- **Premium Design:** A dark, mystical aesthetic with golden accents and smooth 3D card animations.
- **Interactive:** Enter your birth year and flip the card to see your destiny.

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/DigitalGray-Cloud/product-builder.git
   ```
2. Open `index.html` in your web browser.

## 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties, flexbox, 3D transforms, and gradients.
- **JavaScript (ES6+):** Web Components (`<zodiac-card>`), async logic, and dynamic DOM manipulation.
