빠른 입력 매뉴얼

1) 자연어 입력 규칙

키로/킬로/kg는 모두 kg로 인식합니다.

  • 벤치프레스 40 10 5 → 40kg / 10회 / 5세트
  • 벤치프레스 40 5 → 40kg / 10회(기본) / 5세트
  • 벤치프레스 40키로 8회 4세트 → 40kg / 8회 / 4세트

2) 지난 기록 입력

  • 날짜별 상세에서 날짜를 고르면 해당 날짜로 저장됩니다.
  • 상단의 기록 대상 날짜를 확인하고 기록하세요.
  • 오늘로 버튼으로 즉시 오늘 날짜로 복귀할 수 있습니다.

3) 애드센스 고품질 체크리스트

  • 중복/얇은 페이지 대신, 한 주제의 독창적 콘텐츠를 충분히 제공합니다.
  • 사용자가 원하는 정보까지 탐색이 짧고 명확해야 합니다.
  • 광고가 본문보다 더 눈에 띄거나 과도하지 않도록 균형을 유지합니다.
  • 404, 감사/이탈 페이지 등 콘텐츠 가치가 낮은 페이지에는 광고를 피합니다.
  • 검색엔진/광고수익만 노린 키워드 낚시 대신 사용자 의도 중심으로 작성합니다.
