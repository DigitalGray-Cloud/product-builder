# Project Overview
A mystical Fortune Teller web application that provides zodiac-based fortunes (today and yearly) based on the user's birth year.

## Features & Design
- **Multilingual Support:** English (EN) and Korean (KO) language selection.
- **Mystical UI:** A dark, premium aesthetic with golden accents, radial gradients, and subtle "star" background.
- **Custom Web Component:** `ZodiacCard` component for displaying fortunes with a smooth 3D flip animation.
- **Visuals:** High-quality zodiac animal icons with drop shadows and hover effects.
- **Interactive Experience:** Cards auto-flip upon generation and can be manually flipped by the user.
